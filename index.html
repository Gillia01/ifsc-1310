    Progressive Enhancement Writing Assignment

    Progessive enhancement is used in web development as a strategy that can help build a foundation that works for all users. I doesn't matter which device they use, but it improves when a user uses a more upgraded device. Baseline Experience is what starts progressive enhancement. You start by creating a basic version of a website or web application that has core technologies and features that are supported by most browswers. Next is Enhance Features developers can add features by using HTML or JavaScript. These may be interactive elements. Accessibility helps users understand the website easier. Resilience is another core part because the website can still work on older devices.
